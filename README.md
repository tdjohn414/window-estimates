# Estimate Generator

A simple, clean estimate/invoice generator with line items and PDF export.

## Features

- ✅ Add unlimited line items
- ✅ Description, Quantity, Unit, Unit Price per item
- ✅ Auto-calculated Line Total (Qty × Unit Price)
- ✅ Auto-calculated Grand Total (sum of all line totals)
- ✅ Edit any line item
- ✅ Delete line items (updates grand total automatically)
- ✅ PDF export with itemized table
- ✅ Dark mode (auto by time of day + manual toggle)
- ✅ Mobile responsive

## Deploy to Vercel

1. Push to GitHub
2. Go to vercel.com → New Project
3. Import your repository
4. Click Deploy (no environment variables needed!)

## Local Development

```bash
npm install
npm run dev
```

Visit http://localhost:3000
